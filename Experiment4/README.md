# 实验四：DAC 波形发生器

本文件夹用于归档实验四的最终程序和烧录文件。

## 功能说明

- 使用 D/A 转换模块输出不同波形。
- 通过拨码/按键选择波形类型。
- 通过 A/D 读取电位器输入，调节波形输出速度。
- 支持 5 种波形：方波、锯齿波、梯形波、三角波、正弦波。

## 文件说明

- `Test.c`：实验四 C 源程序。
- `Test.hex`：Keil 编译生成的烧录文件。

## 接线与端口说明

- `P2.0` -> DAC 片选 `CS`。
- `P2.1` -> DAC 时钟 `SCLK`。
- `P2.2` -> DAC 数据输入 `DIN`。
- `P1.0` -> 选择方波。
- `P1.1` -> 选择锯齿波。
- `P1.2` -> 选择梯形波。
- `P1.3` -> 选择三角波。
- `P1.4` -> 选择正弦波。
- `ADC(0)` 读取电位器输入，用于改变输出频率。

## 代码讲解要点

- `GetMode()`：读取 P1.0 到 P1.4，判断当前选择哪一种波形。
- `ADC(0)`：读取电位器对应的 A/D 通道。
- `freq = 50 + (uint)ADC(0) * 1950 / 255`：把电位器读数换算成波形周期步数。
- `DAC(uint val)`：把 0 到 1000 的数值转换成 DAC 串行输出。
- `switch(mode)`：根据当前模式分别生成方波、锯齿波、梯形波、三角波和正弦波。
- `sin_tab[256]`：正弦波查表数据，用查表法减少实时计算量。
