# 实验三：电位器电压采集与数码管显示

本文件夹用于归档实验三的最终程序和烧录文件。

## 功能说明

- 读取电位器中端 `Vin` 的模拟电压。
- 通过 A/D 转换模块把模拟量转换为 8 位数字量。
- 将 A/D 数值换算为 0.0V 到 5.0V 左右的电压值。
- 使用两位数码管显示电压，显示格式类似 `2.5`。

## 文件说明

- `test.c`：实验三 C 源程序。
- `test.hex`：Keil 编译生成的烧录文件。
- `实验要求.txt`：实验三原始要求。

## 接线说明

- 电位器中端接到 A/D 模块的 `Vin` 输入端。
- 程序使用 A/D 的 `IN0` 通道，对应代码中的 `XBYTE[0x7000]`。
- 数码管显示模块保持原实验接线，通过串口方式 0 输出段码。

## 代码讲解要点

- `XBYTE[IN0+n]=0`：启动 A/D 转换。
- `addata=XBYTE[IN0+n]`：读取 A/D 转换结果，范围约为 0 到 255。
- `a=a*10/51`：把 A/D 结果换算成扩大 10 倍的电压值，例如 25 表示 2.5V。
- `mydisplay(addata)`：把换算后的电压拆成整数位和小数位，送到数码管显示。
